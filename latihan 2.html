About Text Color

We can use:
  • word
    • such as: yellow
  • red, green, blue
    • such as: rgb(0, 0, 255)
  • hexadecimal
    • with # then 6 digits number and A-F letters, such as: #00FF00
  • hue, saturation, lightness value
    • such as: hsl(360,100%,75%)
  • cyan, magenta, yellow, key(black)
  • alpha transparancy
    • rgba(0,0,255,0.728)
    • hsla(360,100%,75%,0.5)

Let's Learn About CSS Selector

Hierarchically, there are:

  1. id: used on tag with specific id (write # on style)
  2. class: used on tag with specific class (write . on style)
  3. tag: used on tags like heading, paragraph, list, etc

Now Time for Attribute Selector

Giving style to targeted text with the same tag.
You need to type [target] inside style tag (head) and inside the tag itself (body).
You can target for specific target using specific word or just the whole target. Look at the pink background color.
Other than that, you can use :hover on the style tag. Look at the yellow background color that visible with your cursor hovering on the text.
