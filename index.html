Here are some rules for using Heading:

  1. You must and can only have one h1 in one page!;
  2. You can skip which heading you use;
  3. But use h1 to h6 hierarchically;
  4. Don't use heading to get the styles.
    • use the style (css) or text formatting instead, such as:
      • bold to emphasize
      • strong to emphasize more than bold
      • italic for foreign or slang terms
      • subscript like down here
      • superscript like up here
    • because heading has semantic meaning (also bold, strong, and italic)
